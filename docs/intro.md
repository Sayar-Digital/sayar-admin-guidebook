# Sayar Admin Guidebook

This guide is for operations/admin staff using the Admin Portal.

Use this guide to:

- Create merchants
- Complete merchant onboarding
- Triage issues in queues
- Resolve payment/template/checkout health issues
- Track actions with auditability

2. [3. Merchant Lifecycle](03-merchant-lifecycle.md)
3. [4. Onboarding Operations](04-onboarding-operations.md)
4. [5. Needs Attention Queue](05-needs-attention.md)
5. [6. Merchant Health](06-merchant-health.md)

## Scope

This is an operational guide. It avoids deep engineering details and focuses on:

- What each screen is for
- What actions are allowed
- What "good" vs "bad" state means
- What to do next
